# I. Caractérisation des réseaux
## 1. Technologies de transmission
### a. Réseau point à point
Liaison dédié entre deux nœuds de communication

Si $N$ stations doivent communiquer ensemble (full mesh) il faut $\frac{N(N-1)}{2}$ liaisons.

### b. Réseau de diffusion
Il existe un seul canal de communication accessible depuis d'ensemble des nœuds du réseau. 

### c. Acronymes
PAN : Personal Area Network
LAN : Local Area Network
MAN : Metropolitan Area Network (Ville)
WAN : Wide Area Network
